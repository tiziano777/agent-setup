# Arize AX Documentation

## Site Description

Arize AX is an AI engineering platform focused on evaluation and observability. It helps AI engineers and AI product managers develop, evaluate, and observe AI applications and agents. The platform provides comprehensive tools for tracing, evaluation, experimentation, prompt engineering, and production monitoring across the entire AI application lifecycle.

Base URL: https://arize.com/docs
